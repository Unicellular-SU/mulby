# Mulby UI/UX 设计规范

## 1. 设计原则

| 原则 | 描述 |
|------|------|
| **快速** | 最少步骤完成任务，唤起响应 < 200ms |
| **简洁** | 界面清爽，信息层次分明 |
| **一致** | 统一的视觉语言和交互模式 |
| **无干扰** | 用完即走，不打断用户工作流 |
| **键盘优先** | 全程键盘可操作，鼠标为辅助 |

### 2.2 毛玻璃效果 (Glassmorphism)

主窗口采用毛玻璃效果，增强视觉层次感：

```css
.glass-panel {
  background: rgba(255, 255, 255, 0.75);  /* Light */
  /* background: rgba(15, 23, 42, 0.85); */  /* Dark */
  backdrop-filter: blur(16px);
  -webkit-backdrop-filter: blur(16px);
  border: 1px solid rgba(255, 255, 255, 0.2);
}
```

### 2.3 字体系统

**字体选择**: Inter (Google Fonts)

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap');

font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

| 层级 | 字号 | 字重 | 行高 | 用途 |
|-----|-----|-----|-----|-----|
| 标题 | 16px | 600 | 1.5 | 插件名称 |
| 正文 | 14px | 400 | 1.5 | 描述、内容 |
| 辅助 | 12px | 400 | 1.4 | 快捷键、标签 |
| 搜索输入 | 18px | 400 | 1.5 | 搜索框文字 |

## 5. 交互规范

### 5.1 键盘操作

| 按键 | 功能 |
|-----|------|
| `↑` / `↓` | 切换选中项 |
| `Enter` | 执行选中插件 |
| `Esc` | 关闭窗口 / 返回上一级 |
| `Tab` | 切换到插件详情 |
| `⌘/Ctrl + K` | 全局唤起 |
| `⌘/Ctrl + ,` | 打开设置 |

### 5.2 Focus 状态

所有可交互元素必须有可见的 focus 状态：

```css
.focusable:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}
```

### 5.3 Hover 状态

- 使用颜色/透明度变化，避免 scale 导致布局抖动
- 所有可点击元素添加 `cursor: pointer`

## 6. 动画规范

### 6.1 时长标准

| 动画类型 | 时长 | 缓动函数 |
|---------|------|---------|
| 窗口出现 | 150ms | ease-out |
| 窗口消失 | 100ms | ease-in |
| 列表展开 | 200ms | ease-out |
| 选中切换 | 100ms | ease-out |
| Hover 过渡 | 150ms | ease-out |

### 6.2 动画偏移

- 窗口出现：淡入 + 向上移动 8px
- 列表项进入：依次淡入，间隔 30ms

### 6.3 Reduced Motion

尊重用户系统设置，禁用非必要动画：

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 7. 可访问性

### 7.1 对比度要求

| 元素 | 最低对比度 |
|-----|-----------|
| 主文字 | 7:1 (AAA) |
| 次要文字 | 4.5:1 (AA) |
| 图标 | 3:1 |

### 7.2 屏幕阅读器

- 所有图标必须有 `aria-label`
- 动态内容更新使用 `aria-live`
- 列表项包含完整的可读描述

### 7.3 键盘导航

- 所有功能可通过键盘完成
- Focus 顺序符合视觉顺序
- 避免键盘陷阱

## 8. 跨平台适配

### 8.1 平台差异

| 元素 | macOS | Windows | Linux |
|-----|-------|---------|-------|
| 窗口控制 | 隐藏 (无边框) | 隐藏 (无边框) | 隐藏 (无边框) |
| 系统字体回退 | SF Pro | Segoe UI | Ubuntu/Roboto |
| 快捷键修饰 | ⌘ | Ctrl | Ctrl |
| 圆角 | 12px | 8px | 8px |

### 8.2 DPI 适配

- 使用相对单位 (rem) 和 SVG 图标
- 支持 1x、1.5x、2x 缩放
- 图标使用 viewBox 保证缩放清晰

## 9. 实现清单

开发前核对以下项目：

- [ ] 深色主题使用 `#0F172A` 或更深
- [ ] 主文字对比度 ≥ 7:1
- [ ] 所有可点击元素有 `cursor: pointer`
- [ ] Focus 状态清晰可见
- [ ] 动画使用 `ease-out` 缓动
- [ ] 支持 `prefers-reduced-motion`
- [ ] 图标使用 SVG，不用 Emoji
- [ ] Hover 不使用 scale 变换
- [ ] 快捷键根据平台显示正确符号
