# Mulby 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
Mulby - 跨平台插件式效率工具箱

### 1.2 产品定位
面向开发者和效率用户的跨平台桌面应用，通过全局快捷键快速唤起，提供可扩展的插件生态系统。

### 1.3 产品愿景
成为用户桌面效率工具的统一入口，让各类小工具触手可及。

---

## 2. 目标用户

### 2.1 用户画像

| 用户类型 | 特征 | 核心诉求 |
|---------|------|---------|
| 普通效率用户 | 日常办公人员、学生 | 快速使用工具，无需安装多个软件 |
| 开发者用户 | 程序员、技术人员 | JSON处理、编码转换、API调试等开发辅助 |
| 插件开发者 | 有编程能力的用户 | 低门槛开发插件，分享或变现 |
| 企业用户 | 团队、公司 | 内部工具标准化，统一管理分发 |

### 2.2 使用场景

1. **剪贴板处理** - 复制 JSON 后快速格式化/压缩
2. **格式转换** - 图片格式转换、PDF 处理
3. **编码工具** - Base64、URL 编解码、哈希计算
4. **时间工具** - 时间戳转换、时区计算
5. **文本处理** - 正则测试、文本对比、Markdown 预览
6. **开发辅助** - 颜色选择器、UUID 生成、密码生成

---

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 全局快捷键唤起
- 默认快捷键：`Alt+Space` (Windows/Linux) / `Option+Space` (macOS)
- 支持用户自定义快捷键
- 唤起后显示搜索框，支持模糊搜索插件

#### 3.1.2 智能搜索
- 关键词搜索插件
- 自动识别剪贴板内容并推荐相关插件
- 搜索历史记录
- 常用插件置顶

#### 3.1.3 插件管理
- 插件安装/卸载/更新
- 插件启用/禁用
- 插件配置管理
- 本地插件导入

#### 3.1.4 插件商店
- 浏览和搜索插件
- 插件分类展示
- 插件评分和评论
- 插件下载统计

### 3.2 系统功能

#### 3.2.1 设置中心
- 通用设置（开机启动、语言、主题）
- 快捷键设置
- 插件管理
- 数据备份与恢复

#### 3.2.2 系统托盘
- 托盘图标常驻
- 右键菜单快捷操作
- 状态显示

---

## 4. 非功能需求

### 4.1 性能要求
- 冷启动时间 < 3秒
- 快捷键唤起响应 < 200ms
- 内存占用 < 200MB（空闲状态）

### 4.2 兼容性要求
- Windows 10/11
- macOS 10.15+
- Ubuntu 20.04+ / 主流 Linux 发行版

### 4.3 安全要求
- 插件沙箱隔离
- 敏感 API 需用户授权
- 插件代码签名验证（商店插件）

---

## 5. 竞品分析

| 特性 | uTools | Raycast | Alfred | Mulby |
|-----|--------|---------|--------|---------|
| 平台支持 | Win/Mac/Linux | Mac | Mac | Win/Mac/Linux |
| 插件开发语言 | JavaScript | JS/Swift | AppleScript | JS + Python |
| 开源 | 否 | 否 | 否 | 是 |
| 插件 UI | 模板化 | 丰富 | 有限 | 完整前端支持 |
| 价格 | 免费+会员 | 免费+Pro | 付费 | 免费开源 |

### 5.1 差异化优势
1. **完全开源** - 社区驱动，透明可信
2. **双语言支持** - Node.js + Python，降低开发门槛
3. **完整 UI 能力** - 插件可使用任意前端框架
4. **全平台覆盖** - 真正的跨平台体验一致
