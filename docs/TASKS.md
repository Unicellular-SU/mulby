## 开发规范

### 分支策略
- `main` - 稳定版本
- `dev` - 开发分支
- `feature/*` - 功能分支

### 提交规范
```
feat: 新功能
fix: 修复
docs: 文档
refactor: 重构
style: 样式
test: 测试
```
