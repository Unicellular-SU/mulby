# Examples

This folder contains minimal Mulby plugin examples that are meant to be read first and modified second.

Included examples:
- `plugins/hello-clipboard`: read text from input or clipboard, add a prefix, and write it back
- `plugins/timestamp-tools`: copy ISO or Unix timestamps for quick utility workflows

Suggested reading order:
1. `manifest.json`
2. `main.js`
3. `README.md`

These examples are intentionally small. They focus on the runtime contract instead of packaging or build tooling.
