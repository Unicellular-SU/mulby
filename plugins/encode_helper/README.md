# 编码小助手 (encode_helper)

一个功能强大的编码转换工具插件，集成了多种常用的编码、加密和转换功能。

## 功能特性

### 1. 时间转换 ⏰
- 时间戳与日期时间相互转换
- 支持秒级和毫秒级时间戳
- 自动识别输入格式

### 2. Base64编解码 🔢
- Base64编码与解码
- 自动检测输入是编码还是解码
- 支持中文字符

### 3. URL编解码 🔗
- URL编码与解码
- 自动检测输入是编码还是解码
- 处理特殊字符

### 4. 哈希加密 🔐
- 支持多种哈希算法：
  - MD5
  - SHA-1
  - SHA-256
  - SHA-512
- 实时计算哈希值

### 5. Unicode转换 🔤
- Unicode编码与解码
- 自动检测输入是编码还是解码
- 支持\uXXXX格式

### 6. UUID生成 🆔
- 生成标准UUID v4
- 一键生成并复制

### 7. 进制转换 🔢
- 支持二进制、八进制、十进制、十六进制相互转换
- 灵活选择源进制和目标进制
- 自动识别输入格式

### 8. HTML转义 📄
- HTML实体编码与解码
- 自动检测输入是编码还是解码
- 支持所有HTML实体

## 使用方法

### 通过关键词触发
在Intools主界面输入以下关键词触发对应功能：

1. **时间转换**：
   - `时间转换`、`timestamp`、`时间戳`
   - 或直接输入时间戳数字

2. **Base64编解码**：
   - `base64`、`base64编码`、`base64解码`

3. **URL编解码**：
   - `url编码`、`url解码`、`urlencode`、`urldecode`

4. **哈希加密**：
   - `哈希`、`md5`、`sha1`、`sha256`

5. **Unicode转换**：
   - `unicode`、`unicode编码`、`unicode解码`

6. **UUID生成**：
   - `uuid`、`生成uuid`、`guid`

7. **进制转换**：
   - `进制转换`、`二进制`、`十六进制`
   - 或直接输入二进制/十六进制数字

8. **HTML转义**：
   - `html转义`、`html编码`、`html解码`

### 界面操作
1. 打开插件后，可以通过顶部的标签页切换不同功能
2. 在输入框中输入要处理的内容
3. 点击"转换"按钮进行处理
4. 结果会自动复制到剪贴板并显示在输出框中
5. 可以使用"清空"按钮清除输入

## 技术特点

- **智能识别**：自动检测输入内容类型，智能选择编码或解码
- **实时转换**：输入时实时计算并显示结果
- **一键复制**：转换结果自动复制到剪贴板
- **主题适配**：支持亮色和暗色主题
- **响应式设计**：界面适配不同窗口大小

## 安装方法

1. 将插件文件夹复制到Intools的插件目录
2. 重启Intools或重新加载插件
3. 在插件管理中启用"编码小助手"

```bash

npm install

# 开发模式
npm run dev

# 构建插件
npm run build

# 打包插件
npm run pack
```

## 注意事项

1. 哈希计算功能在浏览器环境中使用Web Crypto API
2. 时间转换功能支持常见的时间格式
3. 所有转换结果都会自动复制到剪贴板
4. 插件支持附件处理，可以处理文件内容
