# InTools

跨平台插件式效率工具箱

## 简介

InTools 是一款面向开发者和效率用户的跨平台桌面应用，通过全局快捷键快速唤起，提供可扩展的插件生态系统。

## 特性

- **跨平台支持** - Windows、macOS、Linux
- **插件式架构** - 支持 Node.js 和 Python 开发插件
- **快捷键唤起** - 全局快捷键快速访问
- **丰富的系统 API** - 剪贴板、文件系统、通知等
- **插件 UI 支持** - 使用前端框架开发插件界面
